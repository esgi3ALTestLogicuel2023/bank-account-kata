### ABOUT
 Simple banking account app.
 
 We have a schema define to validate our different entities (that simplify validation process )

 A suite of test to verified the app satisfy to requirements:

 * Account deposit,
 * Account Withdrawal,
 * Account Statement
  
### HOW TO TEST

``` 
 yarn test
```
### HOW TO RUN 

``` 
 node index.js
```
